# My Reading List

A personal collection of article summaries and visual diagrams from interesting reads across the web.

## Structure

```
read/                  # Processed articles
  YYYY/
    month-name/
      YYYY-MM-DD-author-article-title.md

to-read/               # Articles queued for processing
```

## Each Article Includes

- **Source** - Link to the original article
- **Author** - Article author
- **Published** - Publication date
- **TLDR** - 1-2 sentence summary
- **Key Takeaways** - Actionable bullet points
- **Summary** - 2-4 paragraph overview
- **Diagram** - Mermaid visualization of key concepts
